# Quickstart Guide: Python Todo App

## Getting Started

1. Ensure you have Python 3.x installed on your system
2. Clone or download the repository
3. Navigate to the project directory
4. Run the application: `python src/main.py`

## Using the Application

1. The application will display a menu with the following options:
   - 1. Add Task
   - 2. View Task List
   - 3. Update Task
   - 4. Delete Task
   - 5. Mark Task as Complete
   - 6. Exit

2. Enter the number corresponding to the action you want to perform
3. Follow the prompts to provide necessary information
4. The application will display the updated task list after each operation

## Example Workflow

1. Select "1. Add Task" and enter a task title
2. Select "2. View Task List" to see your tasks
3. Select "5. Mark Task as Complete" and enter the task ID to mark it complete
4. Continue using the application to manage your tasks
5. Select "6. Exit" when finished

## Error Handling

- If you enter an invalid task ID, the application will notify you
- If you try to add a task with an empty title, the application will notify you
- If you try to perform an operation on a non-existent task, the application will notify you
